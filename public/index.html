What is "Type the alphabet"?
Type the Alphabet is a Progressive Web App which is available in the Play Store and App store as well as accessible through your web browser at the domain "alphabet-typer.com" You can test how fast you can type the alphabet. Compete with your friends and climb up on the leaderboard.
You can see the leaderboard of all the best times of the current Day. You can see the leaderboard of all the best times of the last Day. You can see the leaderboard of all the best times of the last week. You can see the leaderboard of all the best times of the last month. You can see the leaderboard of all the best times of the current Year. And of course you can also see who has the best times at typing the alphabet of all time. Can you get into the top ten of the best times? Just try it out now. One tip: If you press enter you can restart the timer and start again so you wont have to finish the entire alphabet after making a mistake.

Technologies

This app was developed in the JavaScript framework Vue js. For the styling of the app, the css framework bulma was used and different icon are coming from fontawesome The backend to handle the submitted results is written in Go with the echo http framework and writes the received data into a postgres database. The deployment of the app happens through the google cloud service "cloud run" with the help of docker and nginx.
